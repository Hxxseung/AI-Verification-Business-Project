# AI-Verification-Business-Project

### 1. 목표
- 주문 관리 플랫폼을 제작

<br>

### 2. 목적
- **백엔드 프로젝트** : 기획자, 웹디자이너,프론트 엔지니어의 기능/비기능 요구사항을 구체화 할 수 있다
- **팀** **프로젝트** : 백엔드개발 팀의 일원으로 팀원과 협업을 통해 통합된 어플리케이션을 개발할 수 있다
- **AI서비스** : 생성형 인공지능 서비스(API)와 연동하여 어플리케이션에 AI기능을 개발 할 수 있다

<br>
